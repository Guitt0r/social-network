# Social network

Social network write on the MERN stack

## Features:
- Authorization
- Editing profile
- Uploading photo
- Posting smth
- Edit/Delete own posts
- Likes post
- Meet other users

## Technology Stack:
- Node js
- Express Js
- JWT
- MongoDB
- React
- React Bootstrap
- Redux

## Usage

### Env Variables

Create a .env file in server directory, which contains:

```
PORT=5000
DB_URI='your MongoDb url'
JWT_SECRET='your JWT secret'
CLIENT_URL='http://localhost:3000'
```

## Install Dependencies

```
cd server
yarn
```

```
cd ../client
yarn
```

## Run

Run client:
```
yarn start
```

Run backend:
```
yarn serve
```
